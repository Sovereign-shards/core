# Sovereign Shards Core

This is the core codebase for the Sovereign Shards project—a modular, local-first AI agent pipeline designed for efficient operation on consumer hardware.

## 🚀 Quick Start

1. **Clone this repo:**
   ```bash
   git clone https://github.com/sovereign-shards/core.git
   ```
2. **Run the setup script:**
   ```bash
   ./setup.bat
   ```
3. **Try your first tool:**
   ```bash
   python tools/run/hello_world.py
   ```
4. For deeper documentation or onboarding, see the [Sovereign Shards Launch Repository](https://github.com/sovereign-shards/launch).

## 📁 Repository Structure

- `app/` — Core agent and system scripts
- `tools/run/` — Modular developer tools
- `tests/` — Test suite
- `setup.bat` — Environment/setup script
